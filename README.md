# TheShoppies
Shopify front end dev intern challenge 2021. The application allows users to search and nominate movies for the Shoppies.
This application was built using Node.js/express.js, firebase, html, css, JS and more.
Application allows users to nominate moives and share their nominations with others via a uniquely generated url.

This application is currently hosted with firebase at the following url:

https://theshoppies-e7cd3.web.app/
